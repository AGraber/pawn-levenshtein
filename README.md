# levenshtein.inc

https://en.wikipedia.org/wiki/Levenshtein_distance

## Usage

Simply add to your `pawn.json` and include:

```json
{
    "dependencies": ["ScavengeSurvive/levenshtein"]
}
```

```pawn
#include <levenshtein>
```

## Testing

To test, simply run the package:

```bash
sampctl package run
```

Then connect to `localhost:7777`.
